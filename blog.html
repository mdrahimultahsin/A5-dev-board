Question-2: What is the difference between innerHTML, innerText, and textContent ?

Basically innerText,textContent, innerHTML these are usually use to get or set value in an element.But these have difference.

  • innerText :When need to get or set only text(ignoring hidden element) for this use innerText.
  • innerHTML :innerHtml use When need Gets or sets the HTML content of an element with tags or any selector. innerHtml also shows hidden element,comments. But innerText not shows hidden element or text
  • textContent :textContent basically use when need to Gets or sets all text content. Its smiler like innerText but have difference.textContent gets or sets all text content, also from hidden elements. But innerText Not get hidden texts.
